## Longest Common Subsequence Problem
[http://rosalind.info/problems/ba5c/](http://rosalind.info/problems/ba5c/)

**Given**: Two strings.

**Return**: A longest common subsequence of these strings.

Sample Dataset

```
AACCTTGG
ACACTGTGA
```
Sample Output

```
AACTGG
```

because
```
A-ACTG-G-
ACACTGTGA
```
