# Implement TrieMatching
http://rosalind.info/problems/ba9b/

**Given**: A string *Text* and a collection of strings *Patterns*.

**Return**: All starting positions in *Text* where a string from *Patterns* appears as a substring.

Sample Dataset
```
AATCGGGTTCAATCGGGGT
ATCG
GGGT
```
Sample Output
```
1 4 11 15
```
