# Suffix Tree Construction Problem
http://rosalind.info/problems/ba9c/

Construct the suffix tree of a string.

**Given**: A string *Text*.

**Return**: The strings labeling the edges of `SuffixTree(Text)`. (You may return these strings in any order.)

Sample Dataset
```
ATAAATG$
```
Sample Output
```
AAATG$
G$
T
ATG$
TG$
A
A
AAATG$
G$
T
G$
$
```
